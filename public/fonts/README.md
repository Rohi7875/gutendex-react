# Fonts Directory

Place your Montserrat font files here:

- `Montserrat-Regular.woff2`
- `Montserrat-SemiBold.woff2`

You can download these fonts from:
- [Google Fonts](https://fonts.google.com/specimen/Montserrat)
- [Font Squirrel](https://www.fontsquirrel.com/fonts/montserrat)

## Note
The application will use local fonts if available, otherwise it will fallback to Google Fonts CDN.
